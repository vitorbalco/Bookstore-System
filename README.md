# LibrarySystem-CLI 📚

> Sistema de gerenciamento de acervo literário desenvolvido em Python, focado na demonstração de lógica de programação e estruturação de software via terminal.

---

## 🎯 Objetivo do Projeto
Este projeto foi criado para consolidar conhecimentos fundamentais de **Lógica de Programação** e **Desenvolvimento Backend**. O foco principal é a criação de um fluxo de dados robusto e intuitivo, simulando a operação de uma livraria em um ambiente de interface de linha de comando (CLI).

## 🧠 Destaques de Lógica (Logic Highlights)
Como este projeto prioriza a arquitetura lógica, os seguintes pontos foram implementados:

* **Validação de Entradas:** Algoritmos para garantir a integridade dos dados e evitar falhas no sistema.
* **Controle de Fluxo:** Uso estratégico de estruturas condicionais e de repetição para navegação dinâmica.
* **Modularização:** Organização do código para garantir legibilidade e facilitar a manutenção.

## 🚀 Funcionalidades
- [x] Cadastro de novos títulos (Autor, Nome, Categoria).
- [x] Consulta ao catálogo completo de livros.
- [x] Filtros de busca e listagem organizada.
- [x] Interface interativa com feedbacks ao usuário.

## 🛠️ Tecnologias Utilizadas
* **Linguagem:** Python 3.x
* **Ambiente de Desenvolvimento:** PyCharm / VS Code
* **Paradigma:** Programação Estruturada (Foco em funções e fluxo lógico)
